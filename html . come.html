Introduction to CSS

CSS ka full form Cascading Style Sheets hai. CSS ka use webpages ko attractive aur beautiful banane ke liye kiya jata hai. Is ki madad se hum text ka color, font size, background color aur webpage ka design change kar sakte hain.

CSS ka basic formula hota hai: selector { property: value; } Is formula me selector HTML element ko select karta hai, property batati hai kya change karna hai aur value batati hai change kaisa hoga.

Aaj kal har website me CSS ka use hota hai kyun ke ye webpages ko modern aur stylish banati hai. CSS seekhne se hum achi website designing kar sakte hain.
